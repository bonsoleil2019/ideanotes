# 開発判定シート

> プロジェクト実行可否の客観的評価ツール

## 🎯 評価概要

このシートは**スモールスタート原則**に基づき、プロジェクトの開発移行を客観的に判定するためのツールです。

### 判定基準
- **90点以上**: 即座に開発開始推奨
- **70-89点**: **開発開始OK**（スモールスタート原則適用）
- **50-69点**: 課題解決後に開発開始
- **50点未満**: 企画再検討推奨
